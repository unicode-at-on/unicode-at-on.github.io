  • HTML文件外字相容轉換器
    這個簡單的JavaScript程式可以幫助您將日文假名、簡體字之類的外字，轉換成&#xxxxx;的形式，供HTML文件之用。

  • 檔名修改程式
    平台限制：Windows NT/2000/XP/2003
    這個軟體能修改您硬碟中的檔名。若您在安裝Unicode補完計畫前，曾經使用過櫻花輸入法、font.24之類的日文字型作為檔名，因為這些檔名與Unicode不相容，故您應該修改這些檔名。這個程式能夠幫助您快速修改硬碟中的檔名。
    (特別感謝胡克儀先生提供本程式，本程式已直接捆裝在2.30之後版本的Unicode補完計畫中)

  • 以 C_950.NLS 為表的 BIG5 / Unicode 查碼工具 - Ruby 版
    平台限制：
    1. 系統中要有 c_950.nls，或者將其覆製到此程式同路徑亦可，如兩者皆具會以同目錄的 c_950.nls 為主。
    2. Ruby 1.9.3 以上版本適用
    這個簡單的 Ruby 程式可以將您指定的 big5 / unicode 字，找出其在 c_950.nls 對應的 unicode / big5 內碼。
    如果您放的是原生的 c_950.nls，就會查出標準 cp950；如果您放的是補完的 c_950.nls，就會查出補完的對應。

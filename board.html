為了減輕工作人員的負擔，請注意下列事項。
  • 請務必閱讀置頂的 Q & A 文章，裡面其實已經收錄了大部分常見的問題。
  • 請瀏覽過版上的其他討論串（請爬文），其實多數的問題都是有人問過的。
  • 請確定您的問題與 Unicode 補完計畫有關。
  • 日文軟體、簡體軟體還是亂碼，那不是 Unicode 補完計畫的用途，請參考 AppLocale 之類的軟體。
  • 請閱讀您所下載版本之 Unicode 補完計畫的 readme 檔案，確定是否依要求做到，例如下面是經常被忽略的工作：
    • 2.20 beta 版使用者，安裝或解安裝時，均應重開機兩次。(2.30 alpha 版起一次即可。)
    • Windows 98/Me 的 2.30 alpha 使用者，應事先做長檔名備分。
當您爬文後，仍然找不到答案，且確定問題應與 Unicode 補完計畫相關時，歡迎發問。但發問時請遵守下列規定：
  • 不要使用注音文，使用注音文的文章本站將立刻刪除。
  • 不要使用粵語，目前工作人員沒有人看得懂粵語，粵語文章亦立刻刪除。
  • 您必須明確地描述您碰到的狀況，一般的情況下，您需要把這些資料告訴我們，否則我們無法測試， 亦無法回答您的問題。
    • Windows 版本。如 Windows 95、98、ME、2000、XP、2003。
    • Unicode 補完計畫的版本。如 2.20 beta、2.30 alpha 等。
    • 請描述碰到問題的應用軟體，如 Winamp、Word、KKman 等。
    • 儘可能詳細描述狀況，例如只有一句「出現亂碼」，我們無從得知是怎樣的亂碼。
    • 除了上述資訊請務必詳細描述外，請盡可能提供問題畫面。
  • 由於平台支援 Unicode 的程度有限，Windows 95/98/Me 的使用者，您可能還需要告知下列資訊。Windows 2000/XP/2003 使用者則免。
    • 是否有安裝中國海字集。
    • 細明體 (C:\Windows\Fonts\mingliu.ttc) 的檔案大小
為減輕討論區維護負擔，凡使用注音文、粵語，或問題描述極為不清楚的文章，管理員將不知會而直接刪除， 請特別注意。


若您已經了解規定，請 按此進 入討論區。
